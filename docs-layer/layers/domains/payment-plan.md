# 결제/플랜

구독 결제, 플랜 상태 관리, 프로모션 할인. 외부 결제 게이트웨이와 이메일 알림 시스템을 활용.

## 시스템
- **이메일 알림 시스템** — 결제 성공/실패/만료 알림 발송
- **NicePay 게이트웨이** — `backend/srcs/web/foodlogic/nicepay`. 외부 PG사.

## 데이터
- **Payment, Subscription, Plan, Promotion** — 결제 내역, 구독 상태, 요금제 정의, 프로모션 규칙

## 비즈니스 로직
- **결제 후 플랜 활성화** — 결제 성공 → 구독 생성/갱신 → 이메일 발송 트리거
- **플랜 만료 판정** — 주기적 판정 및 만료 알림
- **프로모션 할인 적용** — 결제 시점 할인 계산

## 인터페이스
- **결제 페이지** — `features/payment`
- **플랜 선택, 프로모션 배너, 온보딩 플랜** — `features/plan`, `promotion-event`, `onboarding-plan`

## 주요 연관
- 계정/인증 (플랜 소유 주체)
- 온보딩/가이드 (온보딩 중 플랜 선택)
- 이메일 알림 시스템 (결제 이벤트)
