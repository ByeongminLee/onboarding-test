# 보안/추적

사용자 행동 추적과 보안 감사 로그. 대부분 백그라운드에서 동작.

## 시스템
- **로깅 인프라, 감사 로그** — `backend/srcs/web/foodlogic/security`. 범용 로깅 기반.

## 데이터
- **TrackingEvent** — 행동 이벤트
- **SecurityAudit** — 보안 감사 레코드

## 비즈니스 로직
- **이벤트 추적 규칙** — 어떤 행동을 어떻게 기록할지
- **이상 탐지** — 비정상 패턴 감지
