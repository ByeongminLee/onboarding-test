# 온보딩/가이드

신규 사용자 진입 경험 전반. 위저드, 투어, 샘플 데이터 제공.

## 시스템
(없음)

## 데이터
- **OnboardingState** — 단계 진행 상태
- **SampleStore** — 샘플 매장 데이터
- **TourProgress** — 투어 완료 여부

## 비즈니스 로직
- **온보딩 단계 진행** — 단계 전환 규칙
- **샘플 데이터 주입** — 신규 사용자에게 예시 데이터 제공

## 인터페이스
- **온보딩 위저드** — `features/onboarding`
- **투어** — `features/tour`
- **샘플 스토어, 테스터** — `features/sample-store`, `tester`

## 주요 연관
- 결제/플랜 (온보딩 내 플랜 선택 단계)
- 계정/인증 (가입 직후 온보딩 진입)
