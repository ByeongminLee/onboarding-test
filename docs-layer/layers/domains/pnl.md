# 손익관리

원가와 판매 데이터를 집계해 손익 지표를 산출. (재무제표 아님 — 운영 관점 손익.)

## 시스템
(없음)

## 데이터
- **Finance 스냅샷** — 기간별 손익 스냅샷
- **MenuSales** — 메뉴별 판매 데이터
- **MenuAnalysis** — 분석 결과

## 비즈니스 로직
- **손익 집계** — 매출 - 원가 = 이익
- **판매량 분석** — 메뉴별 판매 추이/기여도

## 인터페이스
- **손익 대시보드** — `features/income-statement`
- **메뉴 판매 분석** — `features/menu-sales`, `menu-analysis`

## 주요 연관
- 원가계산 (원가 입력)
- 결제/플랜 (유료 기능일 가능성)
